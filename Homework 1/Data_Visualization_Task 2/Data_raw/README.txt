How to use 
Import pandas, matplot, seaborn libraries of python.
Data set is provided in clean data folder.
Src folder has source files.

I utilized Python with Matplotlib for data visualization of student performance data. This involved creating visual representations, including gender distribution bar charts, math score histograms, reading score box plots by parental education, a scatter plot for math vs. reading scores, and a stacked bar chart comparing lunch types and test preparation courses to facilitate data analysis
